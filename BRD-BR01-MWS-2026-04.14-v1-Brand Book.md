## 01 Brand Overview

Maker Web Studios is a web design and SEO agency built exclusively for manufacturers and industrial companies in the RGV and US-Mexico border corridor. Founded by Guillermo Aristi -- 16 years in manufacturing operations, GMP facility management, and ISO 9001 implementation -- we don't speak in generic marketing language. We speak manufacturing.

> "Built by a manufacturer. For manufacturers." -- This is not a tagline. It is a positioning statement that must be reflected in every document, email, and marketing asset we produce.

### Brand Personality

| Element | Specification |
|---|---|
| Voice | Direct. Confident. No agency fluff. We sound like a plant manager who also knows SEO -- not a creative director. |
| Tone | Warm but efficient. Bilingual (English / Mexican Spanish). We talk like a trusted colleague, not a vendor. |
| Authority | We lead with manufacturing credibility first, digital expertise second. The floor earns the desk. |
| Positioning | Premium for the RGV. Not the cheapest. Not the flashiest. The most credible option for serious manufacturers. |

## 03 Color Usage Rules

### Proportion Guidelines

| Element | Specification |
|---|---|
| 60% -- Neutral Base | White (#FFFFFF) and Page Gray (#F2F2F2) form the majority of every page. Clean, open, easy to read. |
| 30% -- Dark Anchor | Deep Charcoal (#1C1C1C) grounds the document. Used in section headers, footers, cover pages, and dark tables. |
| 10% -- Orange Accent | Maker Orange (#E85D26) is used sparingly and intentionally. Section labels, CTAs, left-border callouts, key data points. |

### Digital vs. Print

| Element | Specification |
|---|---|
| Screen (Digital PDF, Web) | Use hex values as specified. Orange: #E85D26. All colors render sRGB. |
| Print (Offset / Laser) | Convert to CMYK. Orange approx: C:0 M:65 Y:85 K:0. Charcoal: C:0 M:0 Y:0 K:88. |
| Email | Use inline hex values. Avoid background fills wider than the email column. Orange CTAs only. |
| PowerPoint / Slides | Use theme colors mapped to this palette. Do not use PowerPoint default themes. |

All Spanish-language documents use the same Arial type system. No typeface changes for language -- only copy changes. Maintain identical font sizes, weights, and spacing in Spanish as in English.

## 05 Typography Usage Rules

### Hierarchy Rules

Every document must use the full hierarchy in order. Never skip a level.

| Element | Specification |
|---|---|
| Level 1 -- Cover Title | One per document. Always on dark background. White text. Never used in body pages. |
| Level 2 -- Section H1 | One per major section. Always in section banner (dark background, orange number prefix). |
| Level 3 -- H2 Subhead | 2-4 per section. Bold, near-black. Introduces a distinct sub-topic. |
| Level 4 -- H3 | Optional. Bold, mid-gray. Used for sub-items within a sub-topic. |
| Level 5 -- Section Label | Orange, all-caps, tracked. Used above H2s to categorize content (e.g. 'PRIMARY COLORS'). |
| Body | All paragraph copy. 22pt in documents (11pt equivalent). 1.4-1.6 line-height. |
| Caption | Light gray, 18pt. Footnotes, image captions, version numbers, metadata. |

## 07 Logo & Wordmark Usage

### Current Logo

The Maker Web Studios wordmark uses a bold orange square followed by the company name in white (on dark) or dark (on light). The current version resembles a square icon mark with bottom-anchored text.

> NOTE: The current logo has visual similarity to the Orange Telecom identity (same orange square, white bold text, bottom-anchored layout). This is not a legal crisis at current scale but should be addressed before scaling outreach beyond the RGV. A differentiation update is planned for Version 2 of this brand book.

### Logo Placement Rules

| Element | Specification |
|---|---|
| On Dark Background | White wordmark with orange mark. Used on cover pages, dark headers, dark footers. |
| On Light Background | Dark wordmark with orange mark. Used on white pages, email signatures, proposals. |
| Minimum Size | Never reproduce the logo below 1 inch (1,440 DXA) in width on print. |
| Clear Space | Maintain clear space equal to the height of the 'M' on all four sides. |
| Forbidden | No stretching, rotation, recoloring, or drop shadows. No placement on busy backgrounds. |

### Cold Email Rules

| Element | Specification |
|---|---|
| Format | Plain text. No HTML styling in cold outreach. Feels personal, not broadcast. |
| Language | English for US-side contacts. Mexican Spanish for Spanish-dominant contacts. Never mix mid-email. |
| Subject Line | Lowercase preferred. Specific to recipient's domain (e.g. 'encontre algo en intercast.com'). |
| Signature | Full signature on every email -- name, title, URL, phone. |
| Tone | Spartan. No 'I hope this email finds you well.' Start with an observation about their business. |

### Social Media Standards

| Element | Specification |
|---|---|
| LinkedIn | Primary channel. Post bi-weekly. Maker Orange used in graphic accents. Dark background cards preferred. |
| Profile Bio | "Web design & SEO for RGV manufacturers. Built by a manufacturer -- 16 years on the floor. makerwebstudios.com" |
| Post Format | Text-first posts. Short paragraphs. No hashtag stuffing. 1-3 relevant hashtags max. |
| Graphics | Use standard brand palette. No stock photos of handshakes. Real industrial imagery preferred. |
