# MWS-LT02 - Daily Prospecting Checklist

**Do this every morning. 2-3 hours. No exceptions.**

---

## Before You Start (5 min)

- [ ] Open Instantly dashboard -- check for overnight replies
- [ ] Open Google Sheets pipeline -- review where every active lead stands
- [ ] Respond to any positive replies from yesterday (within 1 hour of seeing them)

---

## Block 1: Cold Email (30 min)

- [ ] Verify cold email campaigns are active and sending
- [ ] Check bounce rate (pause if over 3%)
- [ ] Check reply rate (adjust copy if under 1% after 500 sends)
- [ ] Add new leads to pipeline if list is running low (50-100 per week)
- [ ] Log any replies: positive, negative, unsubscribe

---

## Block 2: Google Maps Prospecting (30 min)

- [ ] Search Google Maps for target manufacturers or service businesses in RGV (pick one industry per day)
- [ ] Find 5-8 businesses with weak or no websites
- [ ] Run PageSpeed Insights on each (screenshot the score)
- [ ] For scores under 50: note the business name, owner if findable, phone, email
- [ ] Add to Google Sheet lead pipeline
- [ ] Send personalized cold email OR call directly with audit results in hand

---

## Block 3: Cold Calling (30 min)

- [ ] Pull 10-15 leads from pipeline (prioritize warm leads first)
- [ ] Call each one -- 15-minute sprint blocks (5 calls per sprint, 2 sprints)
- [ ] Script: "Hey [name], this is Guillermo from Maker Web Studios. I ran a quick test on your website and found some things costing you leads. Got 2 minutes?"
- [ ] Goal: Book a 15-minute discovery call, not sell on the phone
- [ ] Log every call: answered, voicemail, callback, booked, not interested
- [ ] Leave voicemail if no answer (counts as a touch)

---

## Block 4: LinkedIn (15 min)

- [ ] Send 10-15 connection requests to ICP targets (plant managers, owners in manufacturing)
- [ ] Message new connections: casual, no pitch -- "Saw you run [company] in [city]. Sharp operation."
- [ ] Follow up with existing connections who haven't replied (Day 3, Day 7)
- [ ] Comment on 3-5 target prospects' posts (visibility, not selling)

---

## Block 5: Referrals (10 min)

- [ ] Ask yourself: Who do I know that knows someone who needs a website?
- [ ] Text or call ONE person today and ask for an intro
- [ ] If you completed a project recently, ask that client for a referral
- [ ] Script: "Hey [name], I just wrapped up a project and have room for one more client. Know anyone who's been complaining about their website or not getting leads online?"

---

## End of Day (10 min)

- [ ] Update Google Sheets pipeline with all activity from today
- [ ] Count touches: emails sent, calls made, DMs sent, referrals asked
- [ ] Move leads forward in pipeline (replied > called > booked > proposal sent)
- [ ] Identify tomorrow's priority follow-ups
- [ ] Log daily totals:
  - Emails sent: ___
  - Calls made: ___
  - LinkedIn DMs: ___
  - Referrals asked: ___
  - Discovery calls booked: ___

---

## Weekly Targets (Solo Operator -- Realistic)

These targets assume you're also delivering client work. When you have 0-2 active clients, push toward the upper end. When you have 3+ active clients, hold the lower end. Never drop below the minimums.

| Activity | Daily | Weekly |
|----------|-------|--------|
| Cold emails sent (automated) | 50-75 | 250-375 |
| Cold calls made | 10-15 | 50-75 |
| LinkedIn DMs sent | 10-15 | 50-75 |
| Google Maps audits | 5-8 | 25-40 |
| Referrals asked | 1 | 5-7 |
| Discovery calls booked | 1-2 | 5-8 |
| Proposals sent | 1 | 3-5 |
