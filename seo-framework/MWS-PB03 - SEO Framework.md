## Purpose

Comprehensive SEO knowledge base and execution framework for all Maker Web Studios clients. Every topic has its own folder with deliverables, client application notes, and space for checklists/templates/examples as we build them out.

This is the backbone of the GrowthEngine retainer offering. It extends the existing 6-Prompt system (`seo-service-offering.md`) into a full 82-topic operational playbook.

---

## How to Use This Framework

1. **New Client Onboarding** -- Start at `01-foundations/` and `02-research/`. Run keyword research, competitor analysis, and the 6-Prompt audit system.
2. **Website Build** -- Apply `03-site-structure/`, `04-technical-seo/`, and `05-on-page-seo/` during development. Blueprint 1 handles most of this by default.
3. **GrowthEngine Retainer** -- Execute `06-content-seo/`, `07-link-building/`, `10-analytics/` monthly. This is the recurring value.
4. **Advanced Plays** -- `09-ai-seo/`, `14-programmatic-seo/`, and `15-growth/` for established clients ready to scale.
5. **Cross-Border Clients** -- `08-international-seo/` for RGV businesses serving both US and Mexico markets.
