# MWS Agent Arsenal

49 specialized AI agents for Maker Web Studios operations. Sourced from [agency-agents](https://github.com/msitarzewski/agency-agents) (MIT License) + custom agents.

Each agent has a defined personality, workflow, deliverables, and success metrics. Use them as Claude Code skills, prompt context, or reference frameworks.

## Usage

Reference an agent by reading its file and adopting its workflow for a specific task:

```
"Use the Growth Hacker agent workflow to plan our Q2 acquisition strategy"
"Apply the Reality Checker framework before we ship the client site"
"Run through the Senior Project Manager process on this new spec"
```
